# 服务发现

**服务发现**：又名服务注册，后端服务启动时，会自行将服务注册到服务注册中心中，如：consul、eureka、nacos等。此时，注册中心中有该服务的相关信息，包括服务访问地址、服务调用参数等信息。客户端可以通过使用服务名等标识对后端服务进行调用。

当后端服务启动/停止时，将会发送请求告知注册中心，触发注册中心的新增/删除事件
